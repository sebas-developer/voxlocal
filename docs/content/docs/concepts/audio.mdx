VoxLocal handles audio in multiple formats, from raw NumPy arrays to WAV files and PCM bytes.

## Audio Types

### AudioResult

The output of TTS synthesis:

```python
from voxlocal._audio import AudioResult

result: AudioResult  # From vl.speak() or vl.speak_iter()
```

| Property | Type | Description |
|----------|------|-------------|
| `numpy` | `np.ndarray` | Float32 mono samples, range [-1.0, 1.0] |
| `sample_rate` | `int` | Sample rate in Hz (typically 22050) |
| `duration_seconds` | `float` | Duration calculated from samples |
| `bytes` | `bytes` | Complete WAV file bytes |
| `pcm_s16le` | `bytes` | Raw signed 16-bit little-endian PCM |

### AudioChunk

A portable streaming block:

```python
from voxlocal._audio import AudioChunk

chunk: AudioChunk  # From vl.stream()
```

| Property | Type | Description |
|----------|------|-------------|
| `sequence` | `int` | Chunk order (0-indexed) |
| `numpy` | `np.ndarray` | Float32 mono samples |
| `sample_rate` | `int` | Sample rate in Hz |
| `final` | `bool` | True for the last chunk |
| `source_chunk` | `int \| None` | Original engine chunk index |
| `duration_seconds` | `float` | Duration in seconds |
| `pcm_s16le` | `bytes` | Raw PCM16 bytes |

## Audio Processing

### Normalization

All audio is normalized to mono float32:

```python
import numpy as np
from voxlocal._audio import _mono_float32

# Any input format -> mono float32
audio = np.array([0.5, -0.3, 0.8], dtype=np.float32)
normalized = _mono_float32(audio)
```

### PCM16 Encoding

For wire formats and audio devices:

```python
from voxlocal._audio import _pcm_s16le

pcm_bytes = _pcm_s16le(audio)
# Returns bytes: signed 16-bit little-endian
```

The conversion: `clip(audio, -1, 1) * 32767 -> int16`

## Saving Audio

### WAV File

```python
audio = vl.speak("Hello!")
audio.save("output.wav")
```

### WAV Bytes

```python
wav_bytes = audio.bytes
# Use with any WAV-compatible library
```

### Raw PCM

```python
pcm = audio.pcm_s16le
# For streaming over network or custom playback
```

## Wire Format (NDJSON)

For the HTTP streaming API, chunks are serialized as NDJSON:

```python
wire_dict = chunk.to_wire_dict()
```

Format:
```json
{
  "type": "audio_chunk",
  "sequence": 0,
  "sample_rate": 22050,
  "channels": 1,
  "encoding": "pcm_s16le",
  "final": false,
  "source_chunk": 0,
  "audio_base64": "UklGR..."
}
```

## Sample Rates

Different engines may use different sample rates. VoxLocal enforces consistency within a stream:

```python
for chunk in vl.stream("Hello"):
    # All chunks in a stream have the same sample_rate
    assert chunk.sample_rate == first_sample_rate
```

If the sample rate changes unexpectedly, a `SynthesisError` is raised.
