# Third-Party Models And Engines

VoxLocal source code is licensed under the MIT License. VoxLocal does not
redistribute model weights in its Python wheel.

Applications must review the terms of every downloaded engine and model:

- Supertonic: `Supertone/supertonic-3` and the `supertonic` Python package.
- OpenAI Whisper: `openai-whisper` package and Whisper model terms.
- SenseVoice ONNX: `lovemefan/SenseVoice-onnx` repository and upstream
  SenseVoice terms.
- Moonshine: `moonshine-voice` package. The package reports that non-English
  speech models, including the default Spanish model, use the Moonshine
  Community License and may restrict commercial use:
  <https://www.moonshine.ai/license>

These projects are not covered by VoxLocal's MIT license. This notice is not
legal advice.
